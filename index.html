    Configurar Nuvem (Firebase)

    Ao configurar sua própria nuvem grátis no Firebase (Google), o aplicativo sincroniza seus dados em tempo real entre o celular e o computador!

    Passo a passo rápido:
    1. Crie uma conta gratuita em Firebase Console.
    2. Crie um projeto e adicione um app Web (</>).
    3. Copie o objeto firebaseConfig fornecido pelo Firebase.
    4. Ative o Authentication (E-mail/Senha) e o Firestore Database no console do Firebase.
